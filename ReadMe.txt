**********************************************************************************
本项目包含三大块：数据库操作、libevent实现的多线程服务端、libevent的客户端
项目依赖：
BaseLib:自己的C++代码库
libevent：高并发服务框架。
MySQL：提供数据库的API

组成：
server：实现高并发连接，提供数据库访问以及数据管理服务，为客户端的上传下载服务
	支持Windows，Linux。

client：从服务端下载或向服务端上出文件，
	支持Windows，Linux。
ps: 因MySQL在arm上的检查编译比较麻烦，所以没有在arm上测试。故而客户端没有直接
	加入MySQL的API，但可以通过服务端访问数据库。
